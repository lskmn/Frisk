## Buttons
You can use buttons via the following kirbytag: 
`( button: YOUR TEXT url: YOUR URL color: COLOR icon: FONT AWESOME CODE popup: YES/NO) `

These colors are preconfigured:
(button: green url: typography-elements/button color: green)(button: red url: typography-elements/button color: red)(button: blue url: typography-elements/button color:blue)(button: yellow url: typography-elements/button color:yellow) (button: black url: typography-elements/button color:black)(button: main url: typography-elements/button color:main)

Main stands for the main-theme color that you have selected in the panel.

For icons, simply have a look at (link: http://fontawesome.io/icons/ text: what FontAwesome has to offer popup: yes).

(button: Thousands of icons are ready to use url: http://fontawesome.io/icons/ icon: check-circle popup: yes)
